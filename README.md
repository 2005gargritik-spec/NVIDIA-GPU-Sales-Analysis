# NVIDIA GPU Sales Analysis

## Project Overview

This project analyzes NVIDIA GPU sales data using MySQL and Tableau Public.

## Tools Used

* MySQL
* Tableau Public
* CSV

## Business Questions

* Which GPU sells the most units?
* Which GPU generates the highest revenue?
* Which region generates the highest revenue?
* What are the monthly sales trends?

## Key Findings

* RTX 5060 was the top-selling GPU.
* RTX 4090 generated the highest revenue.
* India generated the highest regional revenue.
* Monthly sales peaked around May.
